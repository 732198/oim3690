- Every HTML element is a box: content, padding, border, margin
- box-sizing: border-box makes width include padding/border (more predictable)
- display: block = full width, stacks vertically
- display: inline = flows like text, ignores width/height
- display: inline-block = inline flow but accepts width/height
- <div> = block container, <span> = inline container
- Google Fonts: link in <head>, apply with font-family
- DevTools box model diagram makes spacing visual and easier to debug
