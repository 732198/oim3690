### Basic HTML Tags (quick reference)

- `<!doctype html>`: declares the page is HTML5.
- `<html lang="en">...</html>`: root element; `lang` helps accessibility.
- `<head>...</head>`: metadata (title, meta, links to CSS).
- `<meta charset="utf-8">`: character encoding—put near the top of `head`.
- `<title>My Page</title>`: browser tab title.
- `<link rel="stylesheet" href="styles.css">`: include external CSS.
- `<style>...</style>`: internal CSS.
- `<body>...</body>`: visible content of the page.
- `<h1>…</h1>` to `<h6>…</h6>`: headings, `<h1>` is most important.
- `<p>...</p>`: paragraph.
- `<a href="https://example.com" target="_blank" rel="noopener">Link</a>`: link (opens new tab safely).
- `<img src="photo.jpg" alt="Description">`: image with alt text.
- `<ul><li>Item</li></ul>` and `<ol>`: lists (unordered and ordered).
- `<div>...</div>`: block container.
- `<span>...</span>`: inline container.
- `<form action="/submit"> <label>Email</label> <input type="email" name="email"> <button>Send</button> </form>`: form elements for user input.
- Semantic layout: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`.

Quick tips:
- Always include `alt` on images for accessibility.
- Use semantic tags to improve accessibility and SEO.
- Put visible content inside `body`; metadata belongs in `head`.
