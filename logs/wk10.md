
## What we covered

- **Arrays** — storing multiple values in one variable; methods: `push()`, `pop()`, `unshift()`, `shift()`, `splice()`, `includes()`, `indexOf()`
- **Loops** — `for`, `while`, `for...of`, `forEach` and when to use each
- **Loop patterns** — summing, finding max/min, counting, filtering, searching with `break`

## Key takeaways

- `i++` is shorthand for `i = i + 1` — it's what moves a loop forward; without it you get an infinite loop
- `const` doesn't freeze an array — you can still `push()` and `pop()`, you just can't reassign the variable entirely
- `indexOf()` returns `-1` when not found, not `null` or `false`
- `break` exits the whole loop; `continue` only skips the current iteration
- `forEach` can't use `break` — use `for...of` when you need to exit early
- In `if/else if` chains, most specific condition must come first
