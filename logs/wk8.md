## JavaScript & the DOM - Key Concepts

### DOM Selection
- `document.querySelector()` - selects the first matching element using CSS selector syntax
- `getElementById()` - selects an element by its unique ID

### Event Listeners
- `addEventListener()` - attaches a function to an element that fires when a specific event occurs (e.g., `click`, `keydown`, `submit`)

### DOM Manipulation Methods
- `textContent` - changes the text inside an element
- `innerHTML` - sets or gets HTML content inside an element
- `classList` - adds, removes, or toggles CSS classes on an element
- `appendChild()` - adds a new child element to a parent
